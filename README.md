# PoolController
A custom pool controller for Pentair Pool Pumps and Chlorinators.
Built with a Raspberry Pi, [.NET 10](https://dot.net) and [Uno Platform](https://platform.uno/).

## This is very much a work in progress!


### Parts List

- [Raspberry Pi 4](https://amzn.to/45My8qW)

- [ELEGOO 4 Channel DC 5V Relay Module](https://amzn.to/4qjRv23)
